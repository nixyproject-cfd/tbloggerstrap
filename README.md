## Feature

- Toggle Dark Mode
- Popular posts on the homepage
- Header Hero
- Automatic Table Of Content
- Sticky Ads
- Nav OfCanvas
- Ad layout
- Widget Populer Post
- Widget Label Post (LIST & CLOUD)
- a list of social media icons in the footer
- Sitemap Page

## Add Sitemap Page
